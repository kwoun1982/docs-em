Inline SVG icon from docs-em's curated Lucide set (1.75 stroke). Use for all UI glyphs — never emoji.

```jsx
<Icon name="search" size={16} />
<Icon name="check-circle" size={14} color="var(--signal)" />
<Icon name="alert-triangle" size={14} color="var(--warn)" />
```

Available names are on `Icon.names`. Common: search, send, file-text, database, layers, activity, gauge, check-circle, alert-triangle, x, rotate-cw, shield-check, trending-up/down, cpu, terminal, clock, filter, lock.
