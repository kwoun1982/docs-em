Square icon-only control for toolbars and inline actions. `active` shows a pressed/selected fill.

```jsx
<IconButton icon="rotate-cw" title="재인덱싱" onClick={reindex} />
<IconButton icon="settings-2" variant="outline" />
<IconButton icon="panel-left" active />
```

Variants: `ghost` (default), `outline`. Sizes `sm`/`md`/`lg`.
